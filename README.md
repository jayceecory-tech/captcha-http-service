# 🔐 验证码识别HTTP服务

基于 ddddocr 的在线验证码识别API服务，支持HTTP接口调用。

## ✨ 特性

- 🚀 **简单易用**：提供RESTful API接口
- 🔧 **开箱即用**：无需训练，直接识别常见验证码
- 🌐 **跨平台**：支持Windows/Linux/macOS部署
- ☁️ **云就绪**：适配Railway、Render、Heroku等云平台
- 📊 **详细日志**：完整的请求/响应日志记录
